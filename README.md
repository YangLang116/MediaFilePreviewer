## 插件安装
> 插件已上传官方Plugins仓库，可搜索 MediaFilePreviewer 下载

##插件说明  
> 如果`IDEA`在预览文件时，出现卡死现象，需切换jbr版本，具体请参考：http://iflutter.toolu.cn/content/chapter-9/part-1.html

## 功能清单
- 支持SVG 预览
- 支持WebP 预览
- 支持SVGA 预览
- 支持Lottie 播放控制

## 说明
> 可在Settings -> Tools -> MediaFilePreview 关闭动态图片自动播放。

## 支持缩放  
| 操作  |    快捷键    |
|:---:|:---------:|
| 放大  | shift + w |
| 缩小  | shift + s |
